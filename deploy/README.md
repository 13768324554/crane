Deploy guide
=============

1. 安装 docker>=1.12
2. 安装 docker-compose
3. 请参考下面 Release 目录启动具体版本
4. 通过浏览器访问 http://$ROLEX_IP 即可，用户名：admin@admin.com 密码：adminadmin


## Release

### v1.0.0rc4
3. 启动环境 `ROLEX_IP=X.X.X.X VERSION=v1.0.0rc4 ./deploy.sh`

3. 启动环境 `ROLEX_IP=X.X.X.X VERSION=v1.0.0rc3 ./deploy.sh`
